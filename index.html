Gypsum by aperiodic

A java program that seamlessly integrates projected images with chalkboard lectures

Gypsum is a program that works with OpenCV to allow you to easily integrate projected images with chalkboard lectures. All you need is a computer, a webcam, a projector, and a blackboard. Once you've installed Gypsum, it will walk you through a simple configuration process, and then let you choose the images to project. To project the images, simply draw a rectangle on the board, and then label the rectangle by drawing a shape above it. The number of holes in the shape corresponds to the index of the image projected into the rectangle. For example, a circle is the first image, a circle with a line through it is the second image, etc.

Dependencies

You must have the OpenCV Framework installed on your system. If you're using a Mac, the Installer is included in the Mac Download. If you're using Linux, you know what to do.

Install

Mac: 1. Launch the included OpenCV Installer Package. 2. Drag the Application Binary to your Apps Folder Linux: 1. Install the OpenCV framework. 2. Launch the .jar file.
